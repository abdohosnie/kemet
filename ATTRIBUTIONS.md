## Adding New Assets

If you are contributing new images or media to KEMET, please:

1. Confirm the asset is in the public domain or under a compatible open license (CC0, CC BY, CC BY-SA)
2. Add an entry to this file with the source URL and license
3. Do not submit copyrighted images without explicit written permission from the rights holder
